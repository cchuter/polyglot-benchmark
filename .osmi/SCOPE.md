# Scope: Kindergarten Garden Exercise

## In Scope

- Implement `Garden` type in `kindergarten_garden.go`
- Implement `NewGarden` constructor with full validation
- Implement `Plants` lookup method
- Pass all tests in `kindergarten_garden_test.go`
- Pass `go vet`

## Out of Scope

- Modifying test files
- Modifying `go.mod`
- Adding external dependencies
- Modifying `.meta/` or `.docs/` directories
- Other exercises or languages

## Dependencies

- Go standard library only (`errors`, `sort`, `strings`)
- No external packages required
