# Scope: Bowling Exercise

## In Scope

- Implement `bowling.go` in `go/exercises/practice/bowling/`
- `Game` struct, `NewGame()`, `Roll()`, and `Score()` methods
- All validation logic for pin counts and game state
- Pass all existing test cases in `bowling_test.go` and `cases_test.go`

## Out of Scope

- Modifying test files (`bowling_test.go`, `cases_test.go`)
- Modifying `go.mod`
- Adding new test cases
- Any changes outside the `go/exercises/practice/bowling/` directory

## Dependencies

- No external dependencies (stdlib only: `errors` package)
- Go 1.18 compatible
