# Scope: polyglot-go-bowling

## In Scope

- Implement the bowling game logic in `go/exercises/practice/bowling/bowling.go`.
- Satisfy all test cases in `bowling_test.go` and `cases_test.go`.
- Pass `go vet` without warnings.

## Out of Scope

- Modifying test files (`bowling_test.go`, `cases_test.go`).
- Modifying `go.mod`.
- Implementing solutions for other exercises.
- Adding benchmarks or additional tests.

## Dependencies

- Go standard library only (`errors` package).
- No external packages required.
