# Scope: Book Store Exercise

## In Scope

- Implementing the `Cost` function in `go/exercises/practice/book-store/book_store.go`
- Any helper functions, types, or constants needed within the same file
- Ensuring all existing tests pass (`book_store_test.go` and `cases_test.go`)

## Out of Scope

- Modifying test files (`book_store_test.go`, `cases_test.go`)
- Modifying `go.mod`
- Modifying `.meta/` or `.docs/` files
- Adding new test cases
- Adding external dependencies
- Changes to any files outside `go/exercises/practice/book-store/book_store.go`

## Dependencies

- Go 1.18+ (as specified in `go.mod`)
- Standard library only (no external packages)
- Package name must be `bookstore`
