# Scope: Palindrome Products (Go)

## In Scope

- Implement `Product` struct in `palindrome_products.go`
- Implement `Products` function in `palindrome_products.go`
- Implement a helper to check if a number is palindromic
- Ensure all existing test cases pass
- Ensure `go vet` passes cleanly

## Out of Scope

- Modifying test files (`palindrome_products_test.go`)
- Modifying `go.mod`
- Implementing bonus test cases (negative numbers)
- Performance optimization beyond passing the [100, 999] range in reasonable time
- Changes to any other exercises or files outside the palindrome-products directory

## Dependencies

- Go standard library only (`fmt`, `strconv`)
- No external packages required
- Module: `palindrome` with `go 1.18`
