# Scope: Beer Song Exercise

## In Scope

- Implementing `Verse(n int) (string, error)` in `beer_song.go`
- Implementing `Verses(start, stop int) (string, error)` in `beer_song.go`
- Implementing `Song() string` in `beer_song.go`
- Passing all tests in `beer_song_test.go`
- Proper error handling for invalid inputs

## Out of Scope

- Modifying the test file `beer_song_test.go`
- Modifying `go.mod`
- Adding new files beyond `beer_song.go`
- Implementing bonus point suggestions (polymorphism, etc.)
- Changes to any other exercise in the repository

## Dependencies

- Go standard library only (`fmt`, `bytes` or `strings`)
- No external packages required
- Go 1.18+ compatibility
