# Scope: Beer Song Exercise

## In Scope

- Implementing the three functions (`Verse`, `Verses`, `Song`) in `go/exercises/practice/beer-song/beer_song.go`
- Handling all special cases: verse 0, verse 1, verse 2, and standard verses (3-99)
- Error handling for invalid verse numbers
- Passing all tests in `beer_song_test.go`

## Out of Scope

- Modifying the test file `beer_song_test.go`
- Modifying the `go.mod` file
- Modifying `.meta/` files
- Adding new test cases
- Bonus objectives (polymorphism, deduplication experiments)
- Changes to any other exercise or language directory

## Dependencies

- Go toolchain (1.18+)
- No external packages required (only standard library: `fmt`, `bytes` or `strings`)
- Reference solution available at `.meta/example.go` for verification
