# Scope: polyglot-go-food-chain

## In Scope

- Implement `Verse`, `Verses`, and `Song` functions in `food_chain.go`
- Ensure all tests in `food_chain_test.go` pass
- Ensure `go vet` passes

## Out of Scope

- Modifying test files
- Modifying `go.mod`
- Adding new dependencies
- Changes to any other exercise
- Performance optimization beyond passing benchmarks

## Dependencies

- Go 1.18+ (per go.mod)
- No external packages required
