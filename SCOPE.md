# Scope: Kindergarten Garden Exercise

## In Scope

- Implement `NewGarden` function in `kindergarten_garden.go`
- Implement `Garden` type with `Plants` method
- Handle all error cases tested by the test suite
- Ensure the input `children` slice is not mutated
- Ensure Garden instances are self-contained (no package-level state)

## Out of Scope

- Modifying test files
- Modifying `go.mod`
- Adding external dependencies
- Changing the `.meta/` or `.docs/` directories
- Changes to any other exercises

## Dependencies

- Go standard library only (`errors`, `sort`, `strings`)
- No external packages
