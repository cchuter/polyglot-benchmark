# Scope: Book Store Exercise

## In Scope

- Implement the `Cost` function in `go/exercises/practice/book-store/book_store.go`
- Package name: `bookstore`
- Must pass all 18 test cases in `cases_test.go`
- Must pass `book_store_test.go` (unit test + benchmark)

## Out of Scope

- Modifying test files (`book_store_test.go`, `cases_test.go`)
- Modifying `go.mod`
- Modifying `.meta/` or `.docs/` files
- Adding new files beyond `book_store.go`
- Any changes outside the `go/exercises/practice/book-store/` directory

## Dependencies

- Go 1.18+ (as specified in `go.mod`)
- Standard library only (no external packages needed)
- The `math` and `sort` packages may be used from stdlib
