# Scope: polyglot-go-markdown

## In Scope

- Implement `Render` function in `go/exercises/practice/markdown/markdown.go`
- Support: paragraphs, headings (h1-h6), bold, italic, unordered lists
- Ensure all 17 test cases pass
- Clean, readable, idiomatic Go code

## Out of Scope

- Modifying test files (`markdown_test.go`, `cases_test.go`)
- Modifying `go.mod`
- Adding external dependencies
- Supporting Markdown features not covered by the test cases (e.g., ordered lists, links, images, code blocks)
- Modifying any files outside the `markdown` exercise directory

## Dependencies

- Go standard library only (`fmt`, `strings`)
- No external packages
