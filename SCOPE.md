# Scope: palindrome-products

## In Scope

- Implement the `Product` struct in `palindrome_products.go`
- Implement the `Products` function in `palindrome_products.go`
- Implement a helper function to check if a number is a palindrome
- Ensure all 5 test cases in `palindrome_products_test.go` pass
- Ensure the benchmark runs correctly

## Out of Scope

- Modifying the test file `palindrome_products_test.go`
- Modifying `go.mod`
- Implementing the commented-out bonus test cases (negative numbers)
- Optimizing beyond what's needed to pass all tests
- Adding new test cases
- Changes to any other exercises in the repository

## Dependencies

- Go standard library only (`fmt`, `strconv`)
- No external packages required
- Reference implementation available at `.meta/example.go`
