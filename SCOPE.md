# Scope: DnD Character Generator

## In Scope

- Implementing the `Character` struct in `dnd_character.go`
- Implementing `Modifier()`, `Ability()`, and `GenerateCharacter()` functions
- Any private helper functions needed (e.g., dice rolling)
- Ensuring all existing tests pass

## Out of Scope

- Modifying test files (`dnd_character_test.go`, `cases_test.go`)
- Modifying `go.mod`
- Modifying `.meta/` or `.docs/` directories
- Adding new test files
- Adding external dependencies

## Dependencies

- Go standard library only (`math`, `math/rand`, `slices`)
- No external packages
