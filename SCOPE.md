# Scope: polyglot-go-pig-latin

## In Scope

- Implement the `Sentence` function in `go/exercises/practice/pig-latin/pig_latin.go`
- Handle all four Pig Latin translation rules
- Handle multi-word input sentences
- Pass all existing test cases in `cases_test.go` and `pig_latin_test.go`

## Out of Scope

- Modifying test files (`pig_latin_test.go`, `cases_test.go`)
- Modifying `go.mod`
- Adding new test cases
- Handling uppercase or mixed-case input
- Handling punctuation or non-alphabetic characters
- Performance optimization beyond passing the benchmark
- Changes to any other exercises or files outside the pig-latin directory

## Dependencies

- Go standard library only (`strings` package for word splitting/joining)
- No external packages required
- The `go.mod` specifies Go 1.18+
