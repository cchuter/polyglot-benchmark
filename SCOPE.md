# Scope: Go Alphametics Solver

## In Scope

- Implementing the `Solve` function in `go/exercises/practice/alphametics/alphametics.go`
- Any helper functions, types, or internal logic needed by `Solve` within the same file/package
- Ensuring all existing test cases pass

## Out of Scope

- Modifying test files (`alphametics_test.go`, `cases_test.go`)
- Modifying `go.mod`
- Modifying `.meta/` files
- Modifying `.docs/` files
- Changes to any other language exercises (python, javascript, java, cpp, rust)
- Adding new test cases
- Adding external dependencies

## Dependencies

- Go 1.18+ toolchain (as specified in `go.mod`)
- No external packages required; standard library only (`errors`, `strings`, `unicode` are sufficient)
- Reference implementation available at `.meta/example.go` for correctness validation
