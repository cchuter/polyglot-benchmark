# Scope: Crypto Square Exercise

## In Scope

- Implementing the `Encode` function in `go/exercises/practice/crypto-square/crypto_square.go`
- All logic needed: normalization, rectangle sizing, column reading, output formatting
- Passing all existing tests in `crypto_square_test.go`
- Passing the benchmark in `crypto_square_test.go`

## Out of Scope

- Modifying test files (`crypto_square_test.go`)
- Modifying metadata files (`.meta/config.json`, `.meta/tests.toml`, `.meta/example.go`)
- Modifying documentation (`.docs/instructions.md`)
- Modifying `go.mod`
- Adding new files beyond the solution file
- Implementing exercises in other languages (python, rust, java, etc.)
- Adding additional exports beyond `Encode`
