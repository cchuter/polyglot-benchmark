# Scope

## In Scope

- Implementing `Verse(n int) (string, error)` in `beer_song.go`
- Implementing `Verses(start, stop int) (string, error)` in `beer_song.go`
- Implementing `Song() string` in `beer_song.go`
- Input validation and error handling for invalid verse numbers

## Out of Scope

- Modifying test files
- Modifying `go.mod`
- Adding external dependencies
- Bonus refactoring challenges mentioned in the issue

## Dependencies

- Go standard library only (`fmt`, `bytes` or `strings`)
- No external packages
