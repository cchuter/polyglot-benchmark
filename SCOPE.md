# Scope: Crypto Square

## In Scope

- Implementing the `Encode` function in `go/exercises/practice/crypto-square/crypto_square.go`
- The function must pass all existing tests in `crypto_square_test.go`

## Out of Scope

- Modifying test files (`crypto_square_test.go`)
- Modifying `go.mod`
- Implementing a `Decode` function
- Changes to any other exercise or language directory
- Adding new dependencies

## Dependencies

- Go standard library only (`math`, `strings`, `unicode`)
- No external packages
